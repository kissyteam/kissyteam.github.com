.. module:: scroll-view

Scroll View
===================================

|  让容器可以滚动

.. note::

    使用时直接引入 scroll-view 即可，会自动判断是否是移动浏览器，以确定是否使用 drag

    .. code-block:: javascript

        KISSY.use('scroll-view', function(S, ScrollView) {
            // use scroll-view
        });
