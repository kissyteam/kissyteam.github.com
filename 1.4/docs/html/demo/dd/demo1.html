Draggable & Proxy 使用示例¶

组织HTML

<div class="box" id="test-drag">
    <s class="box-tp"><b></b></s>
    <div class="box-hd">
        <h3>Drag Me</h3>
    </div>
    <div class="box-bd">
        鼠标拖动
    </div>
    <s class="box-bt"><b></b></s>
</div>

设置代理节点样式

.ks-dd-proxy {
    opacity:0.6;
    *filter:alpha(opacity=60);
}

初始化 draggable 对象

KISSY.use("dd",function(S,DD){
    var drag=new DD.Draggable({
        node:'#test-drag',
        cursor:'move',
        move:true
    });
});

插入 Proxy 插件

drag.plug(Proxy);
