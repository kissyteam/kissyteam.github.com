Editor.Commands.foreColor()¶
foreColor(color)
对选区设置字体颜色插件
Parameters:color (String) – 颜色的 16 进制字符串。例如 ‘#ffffff’

Note

调用命令方式

editor.execCommand("foreColor",'#ffffffff'); // 设置选区字体颜色为白色
