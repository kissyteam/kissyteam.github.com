mvc.sync¶

KISSY.use('mvc',function(S,mvc){
    // use mvc.sync
});
同步函数.
mvc.sync()¶
sync( self , method , opts )
集合或模型发生改变时调用的函数
Parameters:
  • self (Model|Collection) – 当前动作的所处对象
  • method (String) –

    同步的具体动作，包括

    1. create ：创建动作
    2. update ：更新动作
    3. delete ：删除动作
    4. read ：读取动作
  • opts (Object) –

    控制选项，包括

    mvc.sync.opts.success¶

    操作成功后的回调

    mvc.sync.opts.error¶

    操作失败后的回调

    mvc.sync.opts.complete¶

    操作完成后的回调（成功或失败）

Note

系统自带的该函数会自动发请求给模型或集合对应的 url 函数返回的值，如果有特殊需要，可以进行覆盖
