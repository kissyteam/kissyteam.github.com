Class Detail¶

class overlay.Popup¶
Popup (config)
xclass: ‘popup’
  • 继承自 Overlay , 包含其全部配置,属性,方法,事件.
Parameters:config (Object) – 配置项, 详细见下方 Configs Detail .

Note

使用前请加上初始样式

<style>
    .ks-overlay {
        position:absolute;
        left:-9999px;
        top:-9999px;
    }
    .ks-overlay-hidden {
        visibility: hidden;
    }

    .ks-overlay-mask-hidden {
        display: none;
    }

    .ks-overlay-shown {
        visibility: visible;
    }

    .ks-overlay-mask-shown{
        display: block;
    }
</style>

Configs Detail¶

Popup.prototype.trigger¶

{String|HTMLElement[]|KISSY.NodeList} - 触点集合

Popup.prototype.triggerType¶

{String} - 可选, 默认为 ‘click’, 触发类型, 可选 ‘click’, ‘mouse’.

Popup.prototype.mouseDelay¶

{Number} - 单位秒. 可选, triggerType 为 mouse 时, Popup 显示的延迟时间, 默认为 0.1.

Popup.prototype.toggle¶

{Boolean} - 可选, triggerType 为 click 时, Popup 是否有toggle功能，默认为false，不开启

Note

KISSY 1.4 srcNode 初始化组件时必须要求内容节点必须包含类名 ks-popup-content (这里 ks- 为 prefixCls)，例如

<div class="ks-popup" id='t'>
    <div class="ks-popup-content">

    </div>
</div>

<script>
    new Overlay.Popup({
        srcNode:'#t'
    });
</script>
