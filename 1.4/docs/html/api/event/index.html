  • focusout

  • singleTap
  • tap

  • rotateStart
  • rotate
  • rotateEnd

  • pinch
  • pinchEnd

平台统一¶

为了兼容移动与pc， kissy event 提供手势事件的枚举：

Event.Gesture.start¶

手势开始事件

Event.Gesture.move¶

手势进行事件

Event.Gesture.end¶

手势结束事件

Event.Gesture.tap¶

单击事件

Event.Gesture.doubleTap¶

双击事件

Note

手势进行事件在支持触屏的设备和只支持鼠标的设备表现不同。

触屏设备中（pad,ie10,ie11）只有在手势开始事件触发后才会开始手势进行事件。

而支持鼠标的设备（chrome pc，ie<19）则鼠标移动就会开始手势进行事件。
