Class Detail¶

class menubutton.MenuButton¶
MenuButton (config)
xclass: ‘menubutton’
  • 继承自 Button , 包含其全部配置,属性,方法,事件.
Parameters:config (Object) – 配置项，详细见下方 Configs Detail .

Note

从已有元素实例 MenuButton 对象时 , 元素节点标明类名 {prefixCls}menu-button . prefixCls 为配置的类名前缀，默认为 ks- .

MenuButton.config.matchElWidth¶

{Boolean} - 是否下拉菜单和按钮宽度一致。默认 true.
