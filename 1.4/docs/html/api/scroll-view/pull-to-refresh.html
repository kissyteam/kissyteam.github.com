Class Details¶

class scroll-view.PullToRefresh¶
PullToRefresh()
扩展自 Base ，包含其全部配置,属性,方法,事件.
插件 id ：scroll-view/plugin/pull-to-refresh

Configs Details¶

scroll-view.pullingHtml¶
下拉时提示内容 {String}
默认 ‘Pull down to refresh...’
scroll-view.releasingHtml¶
释放操作的提示内容 {String}
默认 ‘release to refresh...’
scroll-view.loadingHtml¶
加载内容时的提示内容 {String}
默认 loading...
scroll-view.loadFn()¶
loadFn(callback)
加载内容的具体函数实现，自行实现，如果不给出，则无操作，只完成动画
callback参数在是组件内部传进来的，在加载完事之后执行这个callback即可
Parameters:callback ({Function}) – 加载完成的callback 
function loadFn(callback) {
    //load something
    //...
    //load finish
    callback();
}

使用示例¶

KISSY.use('scroll-view, scroll-view/plugin/pull-to-refresh', function (S, ScrollView, PullToRefresh) {
    window.scrollView = new ScrollView({
        srcNode: '#wrapper',
        plugins: [
            new PullToRefresh({
                loadFn: function (callback) {
                    setTimeout(function () {
                        scrollView.get('contentEl').prepend('<p>line ' + i++ + '</p>');
                        scrollView.sync();
                        callback();
                    }, 500);
                }
            })
        ]
    }).render();
}
