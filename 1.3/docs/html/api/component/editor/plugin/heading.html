Commands Details¶

Editor.Commands.heading()¶
heading(tag)
对选区设置或取消标题的插件
支持 queryCommandValue，返回选区设置标题
Parameters:tag (String) – 标题的标签名称。例如 ‘h1’,’p’

Note

调用命令方式

editor.execCommand("heading",'h1');
