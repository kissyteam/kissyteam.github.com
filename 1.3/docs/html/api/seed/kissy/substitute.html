KISSY.substitute()¶
String KISSY.substitute (str,o)
将字符串中的占位符替换为对应的键值.
Parameters:
  • str (String) – 包含数据占位符的模板字符串, 占位符用 {} 包起来.
  • o (Object) – 数据
Returns:

将模板和数据结合起来的最终字符串
Return type:

String

例如

var S = KISSY,
str = '{name} is {prop_1} and {prop_2}.',
obj = {name: 'Jack Bauer', prop_1: 'our lord', prop_2: 'savior'};

S.substitute(str, obj); // => 'Jack Bauer is our lord and savior.'
