Class Detail¶

class mvc.View¶
View (config)
  • 继承自 Base , 包含其全部配置,属性,方法,事件.
  • 视图代表一个节点容器，主要用于事件代理
Parameters:
 config (Object) – 配置项.详细见下方 Configs Detail .

Configs Detail¶

View.config.el¶

{String|HTMLDOMNode} - 可选，默认 “<div />” . 选择器或用于 html 字符串或原生节点，用于产生视图容器节点.

View.config.events¶

{Object} - 可选，根节点代理的事件，例如

{
    ".edit":{
        click:"editNote"
    },
    ".new":{
        click:function(e){}
    }
}

那么点击节点容器内带有 edit 样式类的节点会触发当前实例的 editNode 方法，点击带有 new 样式类的节点会触发指定的函数
