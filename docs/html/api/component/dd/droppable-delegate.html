class dd.DroppableDelegate¶
DroppableDelegate (config)
  • 继承自 dd.Droppable , 包含其全部配置,属性,方法,事件.
Parameters:
 config (Object) – 配置项, 详细见下方 Config Attributes Detail

Config Attributes Detail¶

DroppableDelegate.config.container¶

{String | HTMLElement} - 用于委托的容器节点, 所有 Droppable 节点都在其内.

DroppableDelegate.config.selector¶

{String} - 类型选择字符串, 用来获取容器内的 Droppable 节点, 格式为 tag 或 tag.cls 或 .cls.
