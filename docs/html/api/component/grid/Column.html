Class Detail¶

class grid.Column¶
Column (config)
xclass: ‘grid-hd’
  • 继承自 Component.Controller , 包含其全部配置,属性,方法,事件.
Parameters:
 config (Object) – 配置项.

Note

此类用于表格中，不能单独使用，所有的配置项可以当做属性使用

column.set('hide',true);

Configs Detail¶

Column.config.dataIndex¶

{String} - 列对应的数据里的字段名，如： {a : ‘124’}中的‘a’

Column.config.hide¶

{Boolean} - 默认：false，隐藏列

Column.config.id¶

{String} - 默认自动生成，列id ，用于唯一标示列。

Column.config.renderer¶
{Function} - 用于格式化单元格上的数据,函数原型 function(value,record),
  1. value 渲染单元格时的值
  2. record 单元格所在行显示的记录
{
  title : '列一',
  dataIndex : a ,
  renderer : function(value,record){
        if(record.allChange){
            return value * 2;
        }
        return value;
    }
}
Column.config.resizable¶

{Boolean} - 默认：true，列是否可以改变宽度，主要使用允许拖拽列宽，或者列自适应时列自动压缩或者放大

Column.config.sortable¶

{Boolean} - 默认：true ，列是否允许排序

Column.config.sortState¶

{String} - 默认： null,列的排序方向，包含2个值 ‘ASC’,’DESC’

Column.config.title¶

{String} - 默认：&#160，列标题，可以使用html
