Class Detail¶

class overlay.Popup¶
Popup ([container,] config)
Parameters:
 
  • container (String|HTMLElement|KISSY.Node) – 可为’#id’, ‘.class’, DOM对象, KISSY.Node对象, 为空时表示新建
  • config (Object) – 配置项, 详细见下方 Config Attributes Detail .

Note

使用前请加上初始样式

<style>
    .ks-overlay {
        position:absolute;
        left:-9999px;
        top:-9999px;
    }
</style>

Config Attributes Detail¶

与 Overlay 的配置项完全相同, 其他新增配置项如下：
Overlay.prototype.trigger¶

{String|HTMLElement|KISSY.Node} - 触点

Overlay.prototype.triggerType¶

{String} - 可选, 默认为’click’, 触发类型, 可选’click’, ‘mouse’.

New in version 1.3.

Overlay.prototype.mouseDelay¶

{Number} - 可选, triggerType 为 mouse 时, Popup 显示的延迟时间, 默认为 100ms.

New in version 1.3.

Overlay.prototype.toggle¶

{Boolean} - 可选, triggerType 为 click 时, Popup 是否有toggle功能，默认为false，不开启
