class menu.Menu¶
Menu (config)
xclass: ‘menu’
  • 继承自 Controller , 包含其全部配置,属性,方法,事件.
Parameters:config (Object) – 配置项.

Note

从已有元素实例 Menu 对象时 , 元素节点标明类名 {prefixCls}menu . prefixCls 为配置的类名前缀，默认为 ks- .

Menu.prototype.highlightedItem¶

{Menu.Item} - 当前高亮的儿子菜单项.

Hint

menu.set("highlightedItem",null);
