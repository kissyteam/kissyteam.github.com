PopupMenu.config.autoHideOnMouseLeave¶

{boolean} - 默认 false。鼠标移出子菜单时子菜单是否自动隐藏

{number} - 如果设置 autoHideOnMouseLeave 为 true，表示鼠标移出子菜单多长时间后自动隐藏子菜单
