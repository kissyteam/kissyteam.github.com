Class Detail¶

class menu.Item¶
Item (config)
xclass: ‘menuitem’
  • 继承自 Controller , 包含其全部配置,属性,方法,事件.
Parameters:config (Object) – 配置项，详细见下方 Configs Detail .

Note

从已有元素实例 Menu 对象时 , 元素节点标明类名 {prefixCls}menuitem . prefixCls 为配置的类名前缀，默认为 ks- .

Configs Detail¶

Item.config.selectable¶

{Boolean} - 是否可以被选择，可以的话，单击会添加指定 class 到根节点.

Item.config.checkable¶

{Boolean} - 是否可以被多选，可以的话，单击会添加或删除根节点的指定 class .

Item.config.value¶

{*} - 可选，该菜单项的值.

Item.config.content¶

{String} - 该菜单项的显示内容.
