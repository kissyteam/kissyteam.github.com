class tree.Node¶
Node (config)
  • 继承自 ModelControl , 包含其全部配置,属性,方法,事件.
Parameters:
 config (Object) – 配置项.详细见下方 Config Attributes Detail .

Note

从已有元素实例 Node 对象时 , 元素节点标明类名 {prefixCls}tree-item . prefixCls 为配置的类名前缀，默认为 ks- .
