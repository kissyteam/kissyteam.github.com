class UIBase.Box¶
Box (config)
Parameters:
 config (Object) – 配置项，详细见下方 Configs Detail .

Configs Detail¶

UIBase.Box.config.width¶

{Number} - 可选，组件的宽度，单位像素

UIBase.Box.config.height¶

{Number} - 可选，组件的高度，单位像素

UIBase.Box.config.elCls¶

{String} - 可选，附加给组件根节点的样式类

UIBase.Box.config.elStyle¶

{Object} - 可选，附加给组件根节点的内联样式

UIBase.Box.config.elAttrs¶

{Object} - 可选，附加给组件根节点的属性键值对

UIBase.Box.config.elBefore¶

{KISSY.Node} - 可选，组件根节点的渲染到该节点之前.

UIBase.Box.config.render¶

{KISSY.Node} - 默认 S.all(“body”)，组件根节点的渲染为该节点最后一个节点.

UIBase.Box.config.visible¶

{Boolean} - 默认 true ，是否显示

UIBase.Box.config.visibleMode¶

{String} - 默认 “display” ，如何控制显示与否.可选 “display” 与 “visible”

UIBase.Box.config.srcNode¶
{KISSY.Node} - 可选，组件从页面中已存在的该节点中渲染而来.

假设 组件 X 继承自 UIBase 并扩充了 Box

var x=new X({width:12,height:12});
// 渲染为 body 下第一个节点
x.render();
