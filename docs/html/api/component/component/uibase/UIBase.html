Configs Detail¶

UIBase.config.srcNode¶

{KISSY.Node} - 当前组件在 dom 树中已经存在的节点.

UIBase.config.listeners¶

{Object} - 配置组件的事件绑定. 例如

{
    listeners:{
        customEvent:function(e){
            alert(e.type); // => "customEvent"
        }
    }
}

or

{
    listeners:{
        customEvent:{
            fn:function(e){
                // e.type // => customEvent
                // this.xx => 1
            },
            scope:{xx:1}
        }
    }
}
UIBase.config.plugins¶

{Function[]|Object[]} - 插件构造器数组或插件对象数组. 例如

{
    plugins: [ Plugin1,Plugin2 ]
}

or

{
    plugins: [new Plugin1(cfg),new Plugin2(cfg)]
}
