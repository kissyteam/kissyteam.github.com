Command Details¶

Editor.Command.heading()¶
heading(tag)
对选区设置标题的插件
Parameters:
 tag (String) – 标题的标签名称。例如 ‘h1’

Note

调用命令方式

editor.execCommand("heading",'h1');
Editor.Command.queryHeadingActive()¶
queryHeadingActive(elementPath, tag)
对选区设置标题的插件
Parameters:
 
  • elementPath (Editor.ElementPath) – 元素路径
  • tag (String) – 标题的标签名称。例如 ‘h1’

Note

调用命令方式

editor.execCommand("queryHeadingActive",elementPath,'h1');
