focusin¶

原生只有 ie 支持 focusin 事件，而 kissy 对这一事件进行了 兼容性处理 .

定义¶

但一个元素获得焦点或者其子孙元素获得焦点时， focusin 会在该元素上触发（没被子孙元素阻止）。这就是和 focus 事件的区别之处 : 你可以在父元素上监控子元素的 focus 事件，即 focusin 事件支持冒泡.

这个事件常常和 focusout 一起使用.

示例¶

监听 form 中输入域的聚焦情况

<!DOCTYPE html>
<html>
<head>  
    <style>
        .info {
            display:none;
            color:red;
        }
    </style>
    <script src="http://docs.kissyui.com/assets/import-kissy.js"></script>
</head>
<body>
  <form id='f'>
    
    <div>
        <label>姓名：<input/><span class='info'> i get focus</span></label>
    </div>
    
    <div>
        <label>密码：<input/><span class='info'>i get focus</span></label>
    </div>
    
  </form>
<script>
    var $=KISSY.all;
    
    $("#f").on("focusin",function(e){
        var t=$(e.target);
        t.next(".info").stop(true).show().fadeOut();
    });
</script>

</body>
</html>
