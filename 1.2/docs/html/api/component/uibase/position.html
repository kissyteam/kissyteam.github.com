Config Attributes Detail¶

Position.config.zIndex¶

{Number} - 可选, 默认为 9999, 设置组件根节点的 z-index 值.

Position.config.x¶

{Number} - 可选, 组件根节点相对于文档根节点的 x 坐标.

Position.config.y¶

{Number} - 可选, 组件根节点相对于文档根节点的 y 坐标.

Position.config.xy¶

{Array<Number>} - 可选, 相当于将数组第一个元素设置为 x 的值, 将数组的第二个元素设置为 y 的值.
